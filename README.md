## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Python 3.8
* Flask 1.1.x
* SQLALchemy 1.2

### Installation

1. Clone the repo
```sh
git clone https://github.com/github_username/repo_name.git
```

2. Run the application
```bash
python application.py
```
